# use

1. git & ruby & gem install  
`sudo apt install ruby gem`

2. install ruby gems  
`gem install nokogiri mechanize`

3. git clone  
`git clone https://github.com/kakiagengo/scraping.git`

4. edit session id  
chromeのF12 > Networkタブ参照

5. run  
`cd scraping; ./Scraping.rb`
